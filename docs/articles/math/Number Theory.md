# Number Theory

### Mod
- **Basic properties**
```
a ≡ a(mod n)
a ≡ b(mod n)  <==>  b ≡ a(mod n)
a ≡ b(mod n) & b ≡ c(mod n)  =>  a ≡ c(mod n) 
```

- **Basic operations**
```
(a+b) mod n = ((a mod n) + (b mod n)) mod n
(a-b) mod n = ((a mod n) - (b mod n)) mod n
(a*b) mod n = ((a mod n) * (b mod n)) mod n
```

- **Advanced operations**
```
    if a ≡ b(mod n) & c ≡ d(mod n):
        ac ≡ d(mod n)
```
